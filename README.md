2x 27 Resistors
2x 1K Resistors
2x 5.1K Resistors
1x 10K Resistor
1x PinHeader1x03P2.54mm_Vertical
2x PinHeader1x20P2.54mm_Vertical
1x X322512MSB4SI (Quartz Crystal from JLCPCB)
1x TYPE-C-31-M-12 (USB-C Receptacle from JLCPCB)
1x TS-1088-AR02016 (Push Button from JLCPCB)
1x MCP1700x-330xxTT (Fixed Voltage Regulator)
1x W25Q16JVZPIQ TR (Flash Mem Chip)
